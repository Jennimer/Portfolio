Example projects at Arffman

Development of the web applications of Arffman oy, a provider of employment support services: MyArffman for employees, Alku for students, and a portal for public authorities. Application development using agile methods in the cloud, Google applications (Workspace: Docs, Sheets, Sites, etc.), Google Cloud, Javascript, React, Node.js, Zoho CRM system, system integrations using Rest API.

Application and acceptance of registrations

A function on the My Arffman website that retrieves new enrolments for a project or course and displays them in the project details. From the Enrolment field, a modal will open, allowing teachers to review the enrolments and mark them as accepted/rejected.

Listing and editing projects

On the My Arffman site, a function that allows managers to view all projects in Zoho and filter them according to various criteria. The selected project can be viewed and edited, or a new project can be added and changes updated in Zoho.

List and search of employees

Updating the employee view. The application retrieves and lists all employees in Zoho and allows managers to search for employees, as well as easily see missing information and update that information.

Viewing meetings

Arffman start portal home page with a function that shows the student retrieving from Zoho and displays on the student home page the upcoming appointments and opens a modal to view the appointments and data.

Consent to data processing

On first login, a modal opens on the Arffman Start site that asks for consents to process data in different data areas and blocks login if mandatory consents are not given. The consents are stored in Zoho and the Student can view and modify them later.
